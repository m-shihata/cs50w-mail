# Mail
## Installation
1- Clone this repo
2- In the command line...
```
$ cd cs50w-mail
$ python3 -m venv venv
$ pip3 install -r requirements.txt
$ python3 manage.py runserver
```
3- Open your browser and go to localhost:8000
